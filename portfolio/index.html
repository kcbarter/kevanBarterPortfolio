Nitrouz Studioz

Originally a solo project but now a pair project where we are building out a web application via Java Spring Boot. Currently we have the website hooked up to a SQL database to store info to a database. Currently some basic CRUD operations are set up. For this project we are working in sprints to get tasks done.

Seattle Java Users Group

A six month Senior Capstone project were I worked in an agile team of four. We worked to create a website for the a tech meetup group SeaJug so they could better schedule venues, speakers, food sponsors, and after events. I was responsible for helping with fullstack development, writing unit tests, and implementing the Google OAuth login system.

Khalsa Gurmat Center

A three month Project were I worked on an agile team of 4 for the Khalsa Gurmat Center. We were tasked with helping to build a CRM add on to the existing site that helped better keep track of customers and their transactions that was previously done on paper. I was responsible for helping with SQL data base design, frontend/backend validation, and fullstack development.

Meal to Your Door

A five month long project working in an agile team of 4 working in two week sprints. We were tasked with building out a food delivery simulation bot on discord. I was responsible for implementing discord slash commands/button commands, writing JUnit tests, and controller classes to help manipulate/read from the database.

Job Board Website

A two week pair project building out a website. The website we worked on was a simulated JobBoard application. We built out this website using React.js. I was responsible for creating RESTful endpoints with the help of the Axios API, creating function based React components, and creating basic CRUD routes to manipulate/read from our database.

Battleship Game

A two week pair project building out a website. The website we worked on was a battleship game. We built out this website using React.js with the help of the Axios API. I was responsible for implementing react-reducers to help with the state of the board and configuring react.js routes with the help of react-router-dom.
